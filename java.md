#### Access level modifier

1.   default: 封裝內有效
2.   public: 全域有效
3.   private: 類別有效
4.   protected: 封裝及子類別有效

#### Non access modifier

1.   static: 靜態，用於member functions/methods 與 variable
2.   final: 相當於常數化，用於class, methods, variable的實作
3.   abstract: 用來建立抽象類別與方法
4.   synchronized: threads專用，代表同時只能被一個執行序取用
